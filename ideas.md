# Brainstorm de Design - Tozzi Materiais Elétricos

## Contexto
A Tozzi Materiais Elétricos é uma loja especializada em produtos e materiais elétricos. A marca possui uma identidade visual forte com o logo em amarelo e cinza, com um raio como elemento central. O site deve refletir profissionalismo, confiabilidade e energia.

---

## <response>

### Design 1: Industrial Moderno com Energia Dinâmica
**Probabilidade: 0.08**

**Design Movement:** Industrial Design + Modernismo Dinâmico

**Core Principles:**
- Energia visual através de diagonais e formas geométricas angulares (refletindo o raio da logo)
- Hierarquia clara com tipografia em contraste (display bold + body regular)
- Profissionalismo técnico com toques de dinamismo
- Foco em usabilidade e clareza de informação

**Color Philosophy:**
- Amarelo vibrante (#FFC107) como cor primária (raio/energia)
- Cinza escuro (#2C3E50) para base e texto
- Branco (#FFFFFF) para espaço negativo
- Tons de cinza neutro (#E8E8E8, #F5F5F5) para fundos secundários
- Azul escuro (#1A3A52) como cor de destaque complementar
- Intenção: Transmitir confiabilidade técnica com energia e movimento

**Layout Paradigm:**
- Diagonal cuts e ângulos assimétricos em seções (clip-path com polígonos)
- Grid de 12 colunas com quebras assimétricas
- Hero section com imagem de fundo com overlay diagonal
- Produtos em grid 3-4 colunas com cards que se destacam ao hover
- Barra lateral dinâmica com elementos gráficos

**Signature Elements:**
- Linhas diagonais que cortam seções (refletindo o raio)
- Ícones geométricos customizados para categorias de produtos
- Badges com ângulos (não retangulares)
- Setas e indicadores angulares para CTAs

**Interaction Philosophy:**
- Hover effects com movimento diagonal (slide-in de elementos)
- Transições suaves mas perceptíveis (300-400ms)
- Ripple effects em botões
- Cards que se elevam e ganham sombra ao hover

**Animation:**
- Entrance: Elementos entram de forma diagonal (top-left para bottom-right)
- Hover: Movimento sutil com rotação leve (2-3 graus)
- Scroll: Parallax suave em imagens de fundo
- Loading: Animação de raio pulsante

**Typography System:**
- Display: "Poppins Bold" (700) para títulos principais - forte e moderno
- Heading: "Poppins SemiBold" (600) para subtítulos
- Body: "Inter Regular" (400) para texto corrido
- Accent: "Poppins Medium" (500) para labels e CTAs
- Hierarquia: 3.5rem → 2.5rem → 1.5rem → 1rem → 0.875rem

### Design 2: Minimalismo Técnico com Foco em Conteúdo
**Probabilidade: 0.07**

**Design Movement:** Swiss Design + Minimalismo Técnico

**Core Principles:**
- Máxima clareza e legibilidade
- Espaço em branco generoso como elemento principal
- Grid rigoroso e alinhamento perfeito
- Tipografia como protagonista visual

**Color Philosophy:**
- Cinza neutro (#3A3A3A) como cor primária para texto
- Amarelo (#FFC107) em pequenas doses estratégicas (botões, destaques)
- Branco (#FFFFFF) como fundo principal
- Cinza muito claro (#F9F9F9) para seções alternadas
- Preto (#000000) apenas para ênfase máxima
- Intenção: Profissionalismo puro, foco total no produto

**Layout Paradigm:**
- Grid perfeito 12 colunas com espaçamento consistente
- Seções com padding generoso (80-120px vertical)
- Produtos em grid 4 colunas (desktop) com espaço entre
- Navegação horizontal limpa no topo
- Tipografia como elemento visual principal

**Signature Elements:**
- Linhas horizontais finas separando seções
- Números grandes como elementos decorativos (quantidade de produtos, anos de experiência)
- Ícones minimalistas em linha (stroke)
- Espaço em branco estratégico

**Interaction Philosophy:**
- Transições muito sutis (200ms)
- Hover apenas com mudança de cor (sem movimento)
- Foco em acessibilidade e clareza
- Animações mínimas, apenas quando necessário

**Animation:**
- Entrance: Fade-in suave
- Hover: Mudança de cor ou underline
- Scroll: Nenhuma animação, apenas scroll natural
- Loading: Barra de progresso simples

**Typography System:**
- Display: "Playfair Display Bold" (700) para títulos - elegante e clássico
- Heading: "Playfair Display Regular" (400) para subtítulos
- Body: "Inter Regular" (400) para texto corrido
- Accent: "Inter Medium" (500) para labels
- Hierarquia: 4rem → 2.5rem → 1.75rem → 1rem → 0.875rem

## Decisão Final

**Design Escolhido: Industrial Moderno com Energia Dinâmica (Design 1)**

Este design foi selecionado porque:
1. **Alinhamento com a marca:** O uso de diagonais e formas angulares reflete naturalmente o raio presente na logo da Tozzi
2. **Diferenciação:** Em um mercado de lojas de materiais elétricos, a abordagem dinâmica e moderna se destaca
3. **Experiência do usuário:** A hierarquia clara e o layout assimétrico criam uma experiência visual interessante sem comprometer a usabilidade
4. **Escalabilidade:** O sistema é flexível para adicionar novas seções e categorias de produtos
5. **Emoção:** Transmite energia, dinamismo e profissionalismo - exatamente o que uma loja de materiais elétricos deve comunicar

### Implementação
- Tipografia: Poppins para display/headings + Inter para body
- Cores: Amarelo vibrante (#FFC107) + Cinza escuro (#2C3E50) + Branco
- Efeitos: Diagonais, hover com movimento, animações suaves
- Layout: Grid assimétrico com cuts diagonais em seções
